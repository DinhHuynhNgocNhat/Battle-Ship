Detailed Description

Represents the opponent's grid in a battleship game.

The OpponentGrid class manages the tracking of shots fired at the opponent's grid, determines their impact,and keeps track of sunken ships. It provides methods for retrieving grid dimensions, shot information, and sunken ship statistics.
